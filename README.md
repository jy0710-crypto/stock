# 국내 주식 대시보드

실시간 한국 주식 정보 모니터링 대시보드입니다.

## 기능

- 📊 코스피, 코스닥 지수 실시간 추적
- 📈 관심 종목 (현대차, SK하이닉스 등) 모니터링
- 💹 외국인/기관/개인 매매동향 분석
- 📰 주요 뉴스 및 시장 알림

## 기술 스택

- Next.js 14
- React 18
- JavaScript (ES6+)

## 로컬 개발

```bash
# 설치
npm install

# 개발 서버 실행
npm run dev

# http://localhost:3000 에서 확인
```

## 배포

Vercel에 연결되어 있으며, main 브랜치에 푸시하면 자동 배포됩니다.
